# Textanalysator

Detta är ett Python-program för textanalys. Programmet kan:

- Räkna antal ord
- Hitta det längsta ordet
- Visa de mest förekommande orden
- Läsa text från fil eller manuell inmatning
- Spara resultatet i en fil

## Krav

- Python 3.14+
- Virtuellt miljö (venv)
- Se `requirements.txt` för beroenden
